# NapTrackerSDK


iOS 앱에서 사용자 이벤트 및 세션을 추적하기 위한 SDK입니다.


<br>

## ✅ 지원 환경
- iOS 13 이상
- Swift 5.9 이상
- Xcode 16 이상

<br>

## 📦 설치 방법

Swift Package Manager를 통해 설치합니다.

1. Xcode > File > Add Packages...
2. https://github.com/Nasmedia-Tech/iOS-NapTrackerSDK-SPM.git 을 입력하세요. 

<br> 

## 🛠 사용 방법

자세한 연동 방법은 연동 가이드를 참고해주세요.
